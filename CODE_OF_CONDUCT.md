# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful and harassment-free experience for everyone.

## Expected Behavior

* Be respectful
* Be constructive
* Help others learn
* Report issues responsibly

## Unacceptable Behavior

* Harassment
* Discrimination
* Personal attacks
* Spam or abuse

## Enforcement

Project maintainers may remove content or restrict access for violations of this Code of Conduct.
